This is a desktop game of a cat like creature

Current Features:
	Walking left or right on the screen
	Idle state
	being able to drag and move with right mouse click
	reacting and watching to games on windows computers
	custom speech to the contents on screen
	an audio que when speaking
	an audio que when being clicked on by left mouse click

In Progress Features:
	
	particles when clicked
	

Future Features:
	being able to pick up by their tail
	falling state
	sleeping--> can wake up if tapped enough
	Animations for turning around
	Animation for being pet
	state change animations
		idle<-->watch
		Idle<-->walk
		walk<-->watch
